# Vaishnaw Suman — Portfolio Website

A modern, responsive personal portfolio built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean vanilla code.

## 📁 Project Structure

```
portfolio-source/
├── index.html      → Main HTML structure & all sections
├── style.css       → All styling, animations, layout & responsive design
├── script.js       → Interactivity (typewriter, scroll reveal, tilt, counters)
└── README.md       → Project documentation
```

## 🚀 Features

- Animated typewriter effect in hero section
- Scroll-triggered reveal animations
- 3D card tilt effect on project cards
- Animated stat counters
- Cursor glow that follows the mouse
- Sticky navbar with blur on scroll
- Active section highlighting in nav
- Fully responsive (mobile + desktop)
- Contact form with success state

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript (Vanilla ES6+)** — IntersectionObserver, DOM manipulation
- **Google Fonts** — Syne, Inter, JetBrains Mono

## 🏃 How to Run

Just open `index.html` in any browser — no build step, no install needed.

```bash
# Option 1: Direct open
open index.html

# Option 2: Local server (recommended)
npx serve .
# or
python3 -m http.server 3000
```
